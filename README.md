# VALO // Squad Analytics

Frontend-only аналитический dashboard для матчей команды VALORANT. Приложение читает реальные данные из CSV, отбрасывает неполные матчи и пересчитывает показатели при каждом изменении фильтров.

## Запуск

1. Положите CSV в `public/data/team_matches_cleaned_min5_players.csv`.
2. Установите зависимости: `npm install`.
3. Скачайте локальные изображения агентов, карт, ролей и рангов: `npm run fetch-assets`.
4. Запустите dev-сервер: `npm run dev`.
5. Проверьте production-сборку: `npm run build`.

## Акты

Диапазоны `V26:A1`, `V26:A2` и `V26:A3` находятся в `src/config/acts.ts`. Скрипт ассетов также сохраняет ответ Valorant API `/v1/seasons` в каталоге для справки; fallback-диапазоны намеренно лежат в одном файле и легко редактируются вручную.

## Ассеты

`npm run fetch-assets` загружает каталог из публичного Valorant API и сохраняет изображения локально в `public/assets`. Если каталог или отдельная картинка недоступны, интерфейс остаётся рабочим и показывает текстовый fallback-бейдж.
